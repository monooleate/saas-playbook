# 16 — Customer Support & Changelog — Gotchas

> **Status:** Outline.

## Categories

1. **Inbox forgotten when on holiday.** Auto-responder + a co-founder/
   contact's email.
2. **Canned response feels canned.** First sentence personalized;
   rest can be template.
3. **Changelog promises shipped → user finds bug.** Test before
   publish; pull entry if the feature has issues.
4. **Roadmap implies commitment.** Add disclaimer; don't put dates.
5. **Status page lies because manual.** Drive from uptime data.
6. **Feedback widget submits to email; spam follows.** Use a CAPTCHA.

(Real incidents in v0.2.)
