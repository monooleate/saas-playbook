# 16 — Customer Support & Changelog — Checklist

> **Status:** Outline.

## Support

- One inbox for all customer email; SLA target written down.
- 10 canned-response templates for the most common questions.
- In-app "Help" / feedback widget linking to the inbox.
- Sentiment tracking: thumbs up/down at the end of help interactions.

## Changelog

- `/changelog` page on the marketing site.
- Per-release entry with title, summary, image/GIF, tag.
- Link from app footer.
- Weekly cadence target.

## Roadmap

- `/roadmap` page (optional).
- 3 buckets: Now / Next / Later.
- Updated monthly.

## Status page

- Public status page linked from footer + ToS.
- Updated within 5 minutes of an incident.
- Linked to uptime monitor.
