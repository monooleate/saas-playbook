# 22 — Accessibility — Best Practices

> **Status:** Outline.

## Sources

- WCAG 2.2 / W3C ARIA APG.
- axe-core docs.
- Sara Soueidan, Adrian Roselli, Marcy Sutton blogs.
- WebAIM articles.

## Practices

- Native HTML > ARIA. Don't slap `role="button"` on a `<div>`.
- Component libraries (Radix, React Aria) get this right; use them.
- Test with a real screen reader at least quarterly.
- Don't disable focus rings.
- Color contrast is non-negotiable.
- Respect motion-reduction.
