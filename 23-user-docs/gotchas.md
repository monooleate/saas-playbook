# 23 — User Documentation — Gotchas

> **Status:** Outline.

## Categories

1. **Stale screenshots.** UI changed, doc didn't. Most common doc bug.
2. **Article exists but search doesn't find it.** Search index needs
   rebuild on deploy.
3. **Help link in app points at moved URL.** Use stable IDs in URLs.
4. **Docs translated for v1, gone stale.** Either keep translations
   in sync or only translate the truly cornerstone ones.
5. **Doc too long.** Users want one task, get an essay.
6. **Doc too short.** Users want context, get one sentence.

(Real incidents in v0.2.)
