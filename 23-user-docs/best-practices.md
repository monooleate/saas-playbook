# 23 — User Documentation — Best Practices

> **Status:** Outline.

## Sources

- *Every Page is Page One* — Mark Baker.
- Cloudflare / Vercel / Stripe / Linear docs.
- Write the Docs community guides.

## Practices

- Title in user's vocabulary, not yours.
- Every page stands alone (Page One principle).
- Show; don't only tell. GIF + screenshots.
- Update docs as part of the feature PR.
- Track most-visited articles; double down.
- Link from app to specific articles, not the docs root.
