# 23 — Example: Astro Starlight (any stack)

> **Status:** Outline.

`apps/docs/` is an Astro Starlight project. Builds static, served by
Caddy at `/docs`. Pagefind search (no service). MDX articles in
`apps/docs/src/content/docs/`.

Stack-agnostic — same setup works alongside SvelteKit, Next.js, or
Remix.
